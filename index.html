BeaverBird
eliminating untracked behaviour on the Web

BeaverBird will use various tracking techniques to aid sites in following you around on the web.

One such technique is canvas fingerprinting. 

        
var canvasFingerprint = BeaverBird.canvas();
$.post("/fingerprints/fingerprint", canvasFingerPrint)

Track the behaviour of a user over your sites. No cookies. No easy way of evading. 

        
$.get("/fingerprints/" + BeaverBird.canvas(), function(res) {
  // use your awesome new tracking skills to make the user happier than ever!
});

Find out if users are using different hardware, or a different browser. 

          
$.get("/users/42819103/fingerprint" + BeaverBird.canvas(), function(fp) {
  if (BeaverBird.matchCanvas(fp)) {
    // protect the user from horrible attacks! ;)
  }
});
