BeaverBird
eliminating untracked behaviour on the Web
BeaverBird is a sophisticated user tracking library with the aim of making you feel unsafe on the Web.
It currently supports two tracking techniques, and aims at soon providing every technique possible in a client-side library.

Being freely licensed and easy to use, BeaverBird can be integrated into websites with easily, while it still fits into more complex or custom tracking scenarios.

BeaverBird can bring simple and effective tracking to every site.
Features
Canvas Fingerprinting
Font Detection
Unique User Identifiers
All-In-One Tracking
Examples

Let BeaverBird gather a unique id, save it on your server, and attach state to it. 

        
var id = BeaverBird.uid()

$.post("/users/user", id)

Track the behaviour of the "user". No cookies. No easy way of evading. 

        
$.get("/users/" + BeaverBird.uid() + "/interests", function(interests) {
  displayAds(interests)
  // Use your awesome new tracking skills to make the users happier, by following their interests!
});

Find out if users are using different hardware, or a different browser. 

          
$.get("/users/42819103/fingerprint" + BeaverBird.canvas(), function(fp) {
  if (BeaverBird.matchCanvas(fp)) {
    // protect the user from horrible attacks! ;)
  }
});
Installing

Download the latest release from GitHub and include the file 

        
<head>
  <script src="beaverbird.min.js"></script>
</head>

Or use npm 

        
$ npm install --save beaverbird

Then include the file directly or use Browserify 

      
<head>
  <script src="node_modules/beaverbird.min.js"></script>
</head>
